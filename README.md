# Autonomous Linux Infrastructure Discovery Agent

## Mô tả
Agent tự động khám phá và thu thập thông tin về hạ tầng Linux trong môi trường mạng.

## Tính năng chính
- Khám phá thiết bị mạng tự động
- Thu thập thông tin hệ thống chi tiết
- Giao tiếp qua giao thức chuẩn
- Lưu trữ dữ liệu cấu trúc

## Cách bắt đầu
1. Cài đặt yêu cầu hệ thống
2. Chạy script khởi tạo
3. Khởi động agent
